We'll use [Flask](http://flask.pocoo.org) to take
[JSON-RPC](http://www.jsonrpc.org/) requests. It should respond to "ping" with
"pong".

Install Flask to take requests and
[jsonrpcserver](http://jsonrpcserver.readthedocs.io/) to process them:

```sh
$ pip install flask jsonrpcserver
```
Create a `server.py`:

```python
from flask import Flask, request, Response
from jsonrpcserver import methods

app = Flask(__name__)

@methods.add
def ping():
    return 'pong'

@app.route('/', methods=['POST'])
def index():
    req = request.get_data().decode()
    response = methods.dispatch(req)
    return Response(str(response), response.http_status,
                    mimetype='application/json')

if __name__ == '__main__':
    app.run()
```
Start the server:

```sh
$ python server.py
 * Restarting with stat
 * Debugger is active!
 * Debugger pin code: 216-262-392
```

## Client

Use [jsonrpcclient](http://jsonrpcclient.readthedocs.io/) to send requests:

```sh
$ pip install "jsonrpcclient[requests]"
$ python
```
```python
>>> from jsonrpcclient.clients.http_client import HTTPClient
>>> HTTPClient('http://localhost:5000/').request('ping').data.result
'pong'
```
